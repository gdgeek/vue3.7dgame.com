# MRPP 混合现实编程平台 — 产品特点与核心优势

---

## 一、平台概述

MRPP（Mixed Reality Programming Platform，混合现实编程平台）是一款面向 **3D 数字内容创作与编程教育** 的综合性在线平台。平台集成 3D 场景可视化编辑、积木式可视化编程、数字资产管理和校园教学管理等核心功能，广泛适用于 **K12 创客教育、XR 内容创作、数字孪生及企业品牌定制** 等场景。

平台采用 B/S 架构（浏览器/服务器），用户无需安装任何客户端软件，通过主流浏览器即可访问使用，降低了部署和使用门槛。

---

## 二、核心功能特点

### 1. 在线 3D 场景编辑器

平台内置专业级 Web 3D 场景编辑器，支持用户在浏览器中直接进行三维场景的创建、编辑和预览。

- **多格式 3D 模型支持**：兼容主流 3D 模型格式（GLTF/GLB）及体素模型（VOX），支持模型压缩加载，保障大规模场景的流畅体验
- **丰富的多媒体融合**：支持在 3D 场景中嵌入视频、图片、音频、粒子特效等多种媒体资源，构建沉浸式交互体验
- **实时交互能力**：提供 3D 物体点选、轨道环视、动画播放等交互功能，操作直观、响应流畅
- **场景导入/导出**：支持完整场景的一键打包导出与导入，便于场景备份、迁移和跨环境复用

### 2. 积木式可视化编程

集成 Blockly 积木式编程引擎，为编程教育和低代码场景搭建提供强大支持。

- **拖拽式编程**：无需手写代码，通过拖拽积木块即可完成逻辑编排，适合零基础用户
- **多语言界面**：编程界面支持中英日泰等多种语言，满足国际化教学需求
- **数据持久化**：编程作品自动保存与恢复，保护学生创作成果

### 3. 多品牌白标化部署

平台支持一套系统同时服务多个品牌/客户，实现高效的多租户运营。

| 能力 | 说明 |
|------|------|
| 多品牌独立呈现 | 已成功部署 6 个独立品牌站点，各站点拥有独立的名称、Logo、图标和视觉风格 |
| 动态个性化配置 | 品牌配置实时生效，无需重新发布即可调整站点外观与功能 |
| 独立 SEO 优化 | 每个品牌站点拥有独立的搜索引擎优化配置（标题、描述、关键词），提升各站点搜索排名 |
| 品牌级语言锁定 | 可为特定品牌站点锁定指定语言，确保区域化运营一致性 |
| 品牌级风格锁定 | 可为特定品牌站点锁定主题风格，保障品牌视觉统一 |
| 内容平台集成 | 支持对接 WordPress 等内容管理系统，实现资讯、博客等内容发布 |

### 4. 高可用服务架构

平台采用主备双链路架构，保障业务连续性和用户使用体验。

| 能力 | 说明 |
|------|------|
| 主备自动切换 | 当主服务出现异常时，系统自动切换至备用服务，用户无感知 |
| 自动健康恢复 | 持续监测主服务状态，恢复后自动切回，确保最佳访问质量 |
| 会话安全管理 | 用户登录凭证自动维护与刷新，避免频繁重复登录，提升使用体验 |
| 异常统一处理 | 网络异常、权限过期等场景均有完善的提示与处理机制 |

### 5. 精细化权限管理

平台提供多层级、多维度的权限管控体系，满足不同组织架构下的管理需求。

**四级角色体系：**

| 角色 | 适用场景 |
|------|---------|
| 普通用户 | 日常使用：资源管理、项目创作 |
| 管理员 | 团队管理：内容审核、资源调度 |
| 高级管理员 | 组织管理：用户管理、后台运维 |
| 超级管理员 | 平台管理：全权限管控、AI 功能、校园管理 |

**七大权限维度：**
- 页面访问权限 — 控制用户可访问的功能页面
- 内容编辑权限 — 控制用户可编辑的内容范围
- 数据可见性权限 — 控制用户可查看的数据范围
- 角色管理权限 — 控制用户可分配的角色级别
- 作品归属权限 — 控制作品的所有权与协作权
- 功能操作权限 — 按钮级别的细粒度操作控制
- 消息通知权限 — 控制消息的发送与接收范围

### 6. 多语言国际化

平台已完成五大语种的全面适配，支持国际化教学与运营：

| 语言 | 覆盖范围 |
|------|---------|
| 简体中文 | 全平台 |
| 繁体中文 | 全平台 |
| English | 全平台 |
| 日本語 | 全平台 |
| ภาษาไทย (泰语) | 全平台 |

- 语言包按业务模块独立管理，维护便捷、扩展灵活
- 语言切换即时生效，页面标题、描述等 SEO 信息同步更新
- 支持按品牌站点锁定语言，适配区域化运营策略
- 采用按需加载策略，仅加载当前语言资源，不影响系统性能

### 7. 校园管理模块

面向教育场景深度定制的管理模块，覆盖完整的校园组织架构：

| 管理维度 | 功能 |
|---------|------|
| 学校管理 | 学校信息维护、学校列表管理 |
| 教师管理 | 教师信息管理、教学分配 |
| 学生管理 | 学生信息管理、学习跟踪 |
| 班级管理 | 班级创建与管理、师生绑定 |
| 小组管理 | 学习小组划分、协作管理 |

适用于 K12 创客教育、STEAM 编程教学、高校数字媒体课程等教育场景。

### 8. 全类型数字资源管理

平台提供六大类数字资源的统一管理能力：

| 资源类型 | 说明 |
|---------|------|
| 3D 体素模型 | 体素风格 3D 模型的创建、管理与展示 |
| 3D 多边形模型 | 标准 3D 模型的上传、预览与管理 |
| 图片资源 | 支持常见图片格式的上传与管理 |
| 视频资源 | 视频文件的上传、预览与管理 |
| 音频资源 | 音频文件的上传与管理 |
| 粒子特效 | 粒子效果的配置与管理 |

所有资源可在 3D 场景中直接引用，形成完整的"资源管理 → 场景组装 → 作品发布"创作链路。

### 9. 个性化主题系统

平台支持灵活的视觉风格定制，满足不同品牌和用户的审美偏好：

- **明暗模式切换** — 支持浅色/深色两种界面风格，适应不同使用环境
- **自定义品牌色** — 可自由设置平台主色调，匹配品牌视觉标准
- **用户偏好记忆** — 自动保存用户的主题偏好设置
- **品牌级风格锁定** — 可按品牌站点锁定视觉风格，保障品牌一致性
