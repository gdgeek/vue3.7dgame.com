# 预制体管理界面统一化 — 缺陷修复需求文档

## 简介

在全站资源管理界面统一重构过程中，图片、音频、视频、模型（polygen）页面均已迁移至 `StandardPage` 组件体系（`PageActionBar`、`ViewContainer`、`StandardCard`、`PagePagination`、`DetailPanel` 等），并使用 `usePageData` / `useSelection` 组合式函数统一数据与交互逻辑。然而，预制体相关的两个管理页面被遗漏，仍然使用旧版 `CardListPage` + `MrPPCard` / `el-card` 方案，导致 UI 风格与交互能力与其他资源页面不一致。

涉及页面：
- `src/views/phototype/list.vue`（原型列表）
- `src/views/meta/prefabs.vue`（元宇宙预制体列表）

## 缺陷分析

### 当前行为（缺陷）

1.1 当用户访问原型列表页（`phototype/list.vue`）时，系统使用旧版 `CardListPage` + `MrPPCard` 组件渲染页面，与图片、音频、视频、模型页面的 `StandardPage` 风格不一致

1.2 当用户访问元宇宙预制体列表页（`meta/prefabs.vue`）时，系统使用旧版 `CardListPage` + 内联 `el-card`（固定宽度 320px）渲染页面，与其他资源页面的 `StandardPage` 风格不一致

1.3 当用户在原型列表页或预制体列表页尝试搜索资源时，系统不提供搜索输入框，无法按关键词筛选

1.4 当用户在原型列表页或预制体列表页尝试切换视图模式时，系统不提供网格/列表视图切换功能

1.5 当用户在原型列表页或预制体列表页尝试排序资源时，系统不提供排序控件（按时间、按名称）

1.6 当用户在原型列表页或预制体列表页尝试批量操作时，系统不提供多选和批量删除功能

1.7 当用户在原型列表页或预制体列表页查看资源卡片时，系统使用旧版卡片样式，缺少 `StandardCard` 提供的统一选中态、悬停效果和类型图标

### 期望行为（正确）

2.1 当用户访问原型列表页时，系统应当使用 `PageActionBar`、`ViewContainer`、`StandardCard`、`PagePagination` 等 `StandardPage` 组件渲染页面，与图片、音频、视频、模型页面保持一致的视觉风格

2.2 当用户访问元宇宙预制体列表页时，系统应当使用相同的 `StandardPage` 组件体系渲染页面，与其他资源页面保持一致的视觉风格

2.3 当用户在原型列表页或预制体列表页搜索资源时，系统应当在 `PageActionBar` 中提供搜索输入框，支持按关键词筛选预制体

2.4 当用户在原型列表页或预制体列表页切换视图时，系统应当提供网格视图和列表视图的切换功能

2.5 当用户在原型列表页或预制体列表页排序资源时，系统应当提供按创建时间和名称排序的控件

2.6 当用户在原型列表页或预制体列表页进行批量操作时，系统应当支持多选和批量删除功能（通过 `useSelection` 组合式函数）

2.7 当用户在原型列表页或预制体列表页查看资源卡片时，系统应当使用 `StandardCard` 组件，提供统一的选中态、悬停效果和类型图标

### 不变行为（回归防护）

3.1 当用户在原型列表页点击"创建预制体"按钮时，系统应当继续正常跳转至 `/phototype/edit` 页面

3.2 当用户在原型列表页点击"从模型创建"按钮时，系统应当继续正常跳转至 `/phototype/fromModel` 页面

3.3 当用户在原型列表页编辑预制体名称和封面图时，系统应当继续通过编辑对话框（含 `ImageSelector`）正常保存修改

3.4 当用户在原型列表页点击"进入"按钮时，系统应当继续正常跳转至 `/phototype/edit?id=xxx` 编辑页面

3.5 当用户在原型列表页删除预制体时，系统应当继续弹出确认对话框并正常执行删除操作

3.6 当用户在元宇宙预制体列表页点击"创建预制体"按钮时，系统应当继续正常跳转至 `/meta-verse/prefab` 页面

3.7 当用户在元宇宙预制体列表页点击"编辑"按钮时，系统应当继续正常跳转至 `/meta-verse/prefab?id=xxx` 编辑页面

3.8 当用户在元宇宙预制体列表页删除预制体时，系统应当继续弹出确认对话框并正常执行删除操作

3.9 当用户在元宇宙预制体列表页点击预制体卡片缩略图时，系统应当继续正常导航至 `/meta-verse/prefab?id=xxx` 预制体详情页

3.10 当用户为非 Root 角色访问元宇宙预制体列表页时，系统应当继续隐藏创建、编辑、删除按钮（权限控制不变）

3.11 当用户访问图片、音频、视频、模型等已重构页面时，系统应当继续使用 `StandardPage` 组件正常渲染，不受预制体页面重构影响
